Displays a list of blog posts with caching, widget, shortcode, and Gutenberg block support.

== Description ==

This plugin provides multiple ways to display a list of your blog posts:

*   **Gutenberg Block**: Easily add and customize a blog list directly in the block editor.
*   **Shortcode**: Use the `[blog_list]` shortcode in your posts or pages.
*   **Widget**: Add the "Blog List" widget to your sidebars or other widget areas.

Features include post count, category filtering, ordering, pagination, and caching for performance.

== Installation ==

1.  Upload the `blog-list` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Configure settings under Settings > Blog List.
4.  Use the Gutenberg block, shortcode `[blog_list]`, or widget.

== Frequently Asked Questions ==

= How do I use the shortcode? =

You can use `[blog_list posts_per_page="5" category="news" orderby="title" order="ASC"]`.

= How do I use the block? =

Search for "Blog List" in the Gutenberg block editor and add it to your page. Customize options in the block sidebar.
